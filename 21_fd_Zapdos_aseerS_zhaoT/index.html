The Grid

Foundation possesses a gridding system, accessible through the "grid-x" class. You can add as many cells to the grid as you'd like. A grid's length is 12 and each cell's length by default is set to 12. You can alter a cell's length at different browser sizes by using the following class format: size-number where size is small, medium, large and number is positive and is 12 or less.

Try resizing the window to see the cells move in response to the change.

The Buttons

Foundation is equipped with easy button making! You can use the tag button type="button" or the tag a class="button". The former will allows you to specify classes of buttons, while the latter is a more general button. As you can see below, the type buttons look different from the class buttons.

The Menus

Menus are a great way to navigate around a website and Foundation allows for the simple creation of menus. Foundation will provide the basic structure that a menu fulfills, such as linkages to different parts of the website. In addition, there are many stylistic choices that can be made, such as a horizontal vs. a vertical menu. Making a menu is analogous to making an unordered list.

The Forms

Foundation provides support for the creation of forms. Common input methods are of course the text box but others include the dropdown selection box. You are allowed to have placeholders in these text fields by default so then the user knows what to type in. A form is defined with the form tag; the inputs with input tags. In this particular example, the form elements are displayed within a grid.

The Top-Bar

The top-bar exists at the very top of the page (you have to scroll back up)! It is easily supported by Foundation and is a pretty versatile feature. From the example provided on this website, you can see it's able to house not only buttons in a menu-like fashion, but also a search bar.
