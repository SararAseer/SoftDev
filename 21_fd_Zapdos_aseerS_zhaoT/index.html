The Grid

Foundation's possesses a grid system for their webpage. This grid system can be accessed by using the div class "grid-x". Each grid system is compose of cells which can be set to a certain width and therefore take up that much space on the webpage accordingly. The cells are apparent in the examples below, where the text is fitted into "columns" (the cells). Grid also allows for the automatic resizing of these cells, making the webpage adapt to the browser window's settings.

The Buttons

Foundation comes with easy and intuitive ways to make buttons in the front end. You can either use the "a" tag or the "button" tag. Buttons are highly customizable; you can alter the size, appearance, color, functionality, and more with Foundation buttons. The Menus

Menus are a great way to navigate around a website and Foundation allows for the simple creation of menus. Foundation will provide the basic structure that a menu fulfills, such as linkages to different parts of the website. In addition, there are many stylistic choices that can be made, such as a horizontal vs. a vertical menu. Making a menu is analogous to making an unordered list; very simple!
